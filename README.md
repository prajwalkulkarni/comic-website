# Comic World - React + TypeScript + Vite

Comic webapp to browse and search different Marvel comics.

## Installation

Clone the repository to your local machine and run `yarn install`.

After the installation, create a `.env` file and add the following content to it:
`VITE_APP_API_KEY=YOUR_API_KEY_HERE`

Run `yarn run dev` to run the application locally.

## Tech choices

- UI library: ReactJS
- Styling: TailwindCSS
- API - Marvel API
- Bundler - Vite
- Application wide local state management - Context API
- API layer - React Query
